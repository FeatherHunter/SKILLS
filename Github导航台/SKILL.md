# Github导航台 · 技能文档

## 触发词

- `更新文档导航`
- `导航台`
- `docs 导航`

## 核心职责

docs/ 下有多个技能生成的 HTML 子目录：
```
docs/
├── index.html              ← 总览入口（由本技能维护）
├── private-cook/          ← 私家大厨生成
│   └── *.html
└── directory-tree/        ← 目录树生成
    └── *.html
```

本技能负责生成并维护 `docs/index.html`，提供层级浏览体验。

## 工作流程

```
触发「更新文档导航」
    ↓
扫描 docs/ 的直接子目录和文件
    ↓
生成带侧边栏目录树的 index.html
    ↓
如果文件有变化 → commit + push
    ↓
报告结果
```

## 审美规范

taste-skill 标准：
- 纯白背景 #ffffff
- 主文字 #1d1d1f，次要文字 #6b6b6b
- 纯线条 SVG 图标，无 emoji
- Inter + Noto Sans SC 字体
- 大量留白，克制优雅

## 设计结构

```
┌──────────────────────────────────────┐
│ header: 网站标题 + 最后更新时间        │
├────────────┬─────────────────────────┤
│ 侧边栏      │  内容区                  │
│ 整个 docs/ │  点击子目录 → 显示子目录内容│
│ 的目录树    │  点击文件 → 直接打开      │
│            │                         │
└────────────┴─────────────────────────┘
```

- 侧边栏：固定宽度，左侧导航，可折叠
- 内容区：主显示区
- 每个子目录显示为一个可展开的卡片
- 文件按类型分组显示

## 状态管理

```python
current_root: str     # 当前 docs 根目录路径
last_updated: str     # 上次生成时间
```

## 错误处理

| 情况 | 处理 |
|------|------|
| docs/ 目录为空 | 生成带有空状态的 index.html |
| 子目录无文件 | 显示「暂无文件」|
| push 失败 | 报告错误，不删除本地文件 |
