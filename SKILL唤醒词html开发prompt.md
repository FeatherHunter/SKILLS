创建一个单文件 HTML 页面 学习规划师.html，要求：
内容：从SKILL.md、各个md、py脚本中提取所有功能模块，每个模块展示：功能名称、触发词列表、对应的 CLI 命令、简要说明。
分类：沿用技能本身的模块划分
交互：支持按分类tab切换、唤醒词搜索筛选、点击唤醒词看到每个唤醒词底层调用的 CLI 命令是什么、点击CLI命令可以看到CLI 执行流程（这条命令实际改了哪些表、哪些字段、有没有副作用）。
适配：适配手机浏览器
视觉：
【苹果风格】
- 大面积留白，内容不挤
- 纯白/极浅灰底，几乎不用装饰色
- 细线条分隔，不用阴影和色块
- SF Pro 字体（系统字体栈 -apple-system, BlinkMacSystemFont）
- 极少颜色——正文纯黑，交互元素用一个蓝色（#007AFF）就够了
- 圆角小而克制（8-12px），不是大圆角卡片
- 动效克制，hover 微妙变色，不搞花哨动画

本次需求对SKILL.md的新增要求：
在SKILL.md 中 开头单独增加规定1. 该技能的所有优化和变动、脚本的所有变动都要体现在 相应html上
2. 该强制性规定优先级最高
3. 对该技能的所有文件、脚本的任何一行修改都需要明确得到用户的1次确认
