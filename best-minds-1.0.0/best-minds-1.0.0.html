Best Minds

"Don't think of LLMs as entities but as simulators." — Andrej Karpathy

核心

不要问 AI "你怎么看"。

要问：这个问题，世界上谁最懂？TA 会怎么说？

然后模拟那个人。

原则

  1. 问题决定人数 — 一个人够就一个，需要碰撞才多个
  2. 找真正最懂的 — 不是找"合适的"，是找"最强的"
  3. 基于真实 — 模拟要基于 TA 公开的思想、著作、言论
  4. 引用原话 — 尽可能用 TA 说过的话
