# 目录模式 · 技能文档

## 核心交互逻辑

用户在微信里通过文字指令，像操作文件夹一样逐层深入/回退目录，选中文件后下达具体操作指令（发送、分析、修改等）。

---

## 触发与退出

| 操作 | 用户输入 | AI 行为 |
|------|----------|---------|
| 首次唤醒 | `目录模式` | 进入根目录选择 |
| 恢复上下文 | `目录模式` | 回忆当前目录位置，重新显示内容 |
| 返回上一层 | `0` | 退回上级目录 |
| 彻底退出 | `退出目录模式` | 退出导航模式，记住当前目录路径 |

---

## 启动流程（首次唤醒）

AI 输出固定格式：

```
📂 目录模式 · 选择根目录
━━━━━━━━━━━━━━━━━━━━
请选择操作目录：

[1] 工作目录
    D:\2Study\StudyNotes

[2] 日志目录
    D:\2Study\StudyNotes\2026\个人\20260503
    （系统自动以当天日期构建路径，如不存在将报错提醒）

[3] 其他目录
    请输入完整路径

━━━━━━━━━━━━━━━━━━━━
输入数字选择，0退出目录模式
```

### 日期目录构建规则

日志目录格式：`D:\2Study\StudyNotes\{年份}\个人\{年月日}`
- 年份：当前年份（如 2026）
- 月日：YYYYMMDD 格式（如 20260503）

**不存在时**：AI 报错提示「日志目录不存在，请选择其他目录」，不自动创建。

## 导航状态格式（固定格式）

每次目录展示严格遵循此格式，用户形成肌肉记忆：

```
📂 D:\2Study\StudyNotes\2026\个人\
━━━━━━━━━━━━━━━━━━━━
——文件夹——
[1] 01-日记/
[2] 02-周记/
[3] 03-随手记/

——文件——
[4] 2026年度目标.md
[5] 阅读清单.txt

━━━━ 第1页/共3页 ━━━━
上一页 下一页 0返回上一层
━━━━━━━━━━━━━━━━━━━━
输入数字进入文件夹/选中文件，0返回上一层
```

### 格式规则

- **路径显示**：完整路径，不做缩写
- **文件夹在前、文件在后**：两组分别排列，之间用空行分隔
- **每页最多10项**：超出自动分页，支持「上一页」「下一页」
- **当前页标注**：「第X页/共Y页」

## 选中文件后的操作流程

```
你选中了：travel.jpg
━━━━━━━━━━━━━━━━━━━━
请输入指令：（可发送微信、读取内容、分析处理等）
```

用户下达指令，格式：`{数字} {指令}`

示例：
- `3 发给微信` → 将当前目录下的第3项发送到微信
- `5 读取内容` → 读取当前目录下的第5项

AI 根据空格分割：
- 数字部分 → 定位当前目录下的条目
- 其余部分 → 执行对应操作

**AI 困惑时**：主动交互式澄清，不自作主张。

---

## 核心规则

| 规则 | 说明 |
|------|------|
| 固定格式 | 每次目录展示格式统一，用户形成稳定预期 |
| 上下文保持 | 退出目录模式后，AI 内部记住当前目录路径，同一会话内说「目录模式」恢复 |
| 指令不受限 | 导航过程中可随时下达其他指令（查天气、写作等），AI 记住目录位置 |
| 权限禁区 | 无法访问的目录直接通知用户，不卡死 |
| 报错处理 | AI 困惑时主动交互式澄清，不自作主张 |

---

## 状态管理

### AI 内部维护（会话级）

```
current_root: str          # 当前根目录路径
current_path: str          # 当前完整路径
current_page: int          # 当前页码
total_pages: int           # 总页数
navigation_history: list   # 导航历史，用于支持多次0返回
```

### 初始状态

首次启动前，以上状态均为空，说「目录模式」时根据是否有已有状态决定：
- 有状态 → 恢复目录展示
- 无状态 → 进入根目录选择

---

## 分页逻辑

- 每页最多10项（文件夹+文件混合计数上限10）
- 文件夹与文件分别计算，分别排列
- 例如：3个文件夹 + 7个文件 = 1页；3个文件夹 + 8个文件 = 第一页7个文件，第二页1个文件
- 用户输入「下一页」/「上一页」切换，AI 根据已有目录内容响应
