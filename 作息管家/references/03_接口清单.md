# 接口清单

> 本文件列出所有可用的数据库操作接口。**严禁直连数据库，必须通过 CLI 调用。**

---

## schedule_records 操作接口

### 增加记录

```python
add_record_full(
    date,                  # 日期（YYYY-MM-DD）
    time_start,           # 开始时间（HH:MM）
    time_end,             # 结束时间（HH:MM）
    duration_minutes,     # 持续时长（分钟）
    activity,             # 活动描述
    category,             # 分类
    source_contents,       # 消息原文
    source_timestamps,     # 消息时间
    analysis_reasoning     # AI分析推理
)
```
⚠️ **9个字段缺一不可，少传直接报错**

### 更新记录

```python
update_record(record_id, **kwargs)
# record_id: 必填，定位记录
# kwargs: 可选字段 date/time_start/time_end/duration_minutes/
#         activity/category/source_contents/source_timestamps/analysis_reasoning
```

### 查询接口

| 接口 | 说明 |
|------|------|
| `get_last_record_full()` | 获取最后一条完整记录 |
| `get_records_by_date(date_str)` | 获取指定日期所有记录 |
| `get_records_range(start_date, end_date)` | 获取日期范围内所有记录 |
| `has_records_for_date(date_str)` | 检查是否有记录 |

## daily_summary 操作接口

### 增加摘要

```python
add_summary_full(
    date,                         # 日期（YYYY-MM-DD）
    total_sleep_minutes,          # 睡眠总时长
    total_work_minutes,           # 工作总时长
    total_exercise_minutes,        # 运动总时长
    total_commute_minutes,         # 通勤总时长
    total_eating_minutes,          # 餐饮总时长
    total_learning_minutes,         # 学习总时长
    total_entertainment_minutes,    # 娱乐总时长
    total_unknown_minutes           # 未知活动总时长
)
```
⚠️ **9个字段缺一不可，少传直接报错**

⚠️ **前提条件**：该日期的 schedule_records 必须已填满 24 小时

### 更新摘要

```python
update_summary(date, **kwargs)
# date: 必填，定位记录
# kwargs: 可选字段 total_sleep_minutes 等8个minute字段
```

### 查询接口

| 接口 | 说明 |
|------|------|
| `get_daily_summary(date_str)` | 获取每日摘要 |
| `get_summaries_range(start_date, end_date)` | 获取日期范围摘要 |

---

## 游标查询接口（用于 AI 同步）

| 接口 | 说明 |
|------|------|
| `get_messages_for_sync(cursor_datetime_str)` | 获取游标前10条到现在的所有消息 |
| `get_messages_before(before_time_str, limit=10)` | 获取游标前 N 条消息 |
| `get_messages_from(from_time_str)` | 获取某时间后的所有消息 |

## 调用示例

### 写入一条作息记录

```python
from schedule_db import add_record_full

add_record_full(
    date='2026-05-22',
    time_start='08:00',
    time_end='09:30',
    duration_minutes=90,
    activity='工作、写代码',
    category='工作',
    source_contents='在写代码; 优化一下技能; 检查下bug',
    source_timestamps='08:00; 08:30; 09:00',
    analysis_reasoning='消息内容涉及代码编写和技能优化，属于工作活动'
)
```

### 写入每日摘要

```python
from schedule_db import add_summary_full

add_summary_full(
    date='2026-05-22',
    total_sleep_minutes=420,
    total_work_minutes=480,
    total_exercise_minutes=0,
    total_commute_minutes=36,
    total_eating_minutes=90,
    total_learning_minutes=0,
    total_entertainment_minutes=60,
    total_unknown_minutes=174
)
```
