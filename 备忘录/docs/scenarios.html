备忘录技能

全部使用场景与效果演示

📝 笔记管理

场景 1：快速添加笔记
你
帮我记一下，张三的生日是10月3号，放社交夹里。
AI
好的，已记录。
AI 解析意图
→
提取内容 + 分类
→
调用 CLI
$ script/memo_cli.py add "张三生日10月3号" -c social
返回结果
{"status": "ok", "data": {"id": 1, "content": "张三生日10月3号", "category": "social"}, "message": "笔记已添加"}
场景 2：带媒体的笔记
你

AI
小票已存档，分类为记账。
保存图片
→
调用 CLI
$ script/memo_cli.py add "购物小票 2026-05-23" -c finance -m slip_20260523.jpg
返回结果
{"status": "ok", "data": {"id": 2, "content": "购物小票 2026-05-23", "category": "finance"}, "message": "笔记已添加"}

⏰ 定时提醒

场景 9：一次性提醒
你
提醒我明天下午3点打疫苗。
AI
好的，明天下午3点提醒你。
添加笔记
→
获取 id=11
→
设置提醒
$ script/memo_cli.py add "打疫苗"
$ script/memo_cli.py remind 11 --at "2026-05-24 15:00"
效果
明天下午 3:00，cron 任务检测到提醒，通过 QQ 推送：【提醒】2026-05-24 15:00 - 打疫苗
场景 10：每周重复提醒
你
每周五早上9点提醒我还信用卡。
AI
已设置每周五9点提醒还信用卡。
$ script/memo_cli.py add "还信用卡" -c finance
$ script/memo_cli.py remind 12 --repeat-type weekly --rule "5 09:00"
重复规则
weekly "5 09:00" = 每周五(5) 早上 9:00
场景 11：每天重复提醒
你
设置每天8点提醒我喝水。
AI
已设置每天8点提醒喝水。
$ script/memo_cli.py add "喝水"
$ script/memo_cli.py remind 13 --repeat-type daily --rule "08:00"
重复规则
daily "08:00" = 每天早上 8:00
场景 12：每年重复提醒
你
每年12月25号早上10点提醒我圣诞节。
AI
已设置每年12月25日提醒。
$ script/memo_cli.py add "圣诞节"
$ script/memo_cli.py remind 14 --repeat-type yearly --rule "12-25 10:00"
重复规则
yearly "12-25 10:00" = 每年 12 月 25 日 10:00
