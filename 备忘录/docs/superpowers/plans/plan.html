并发安全

配置说明代码
WAL 模式 允许并发读写，提升多进程安全性 PRAGMA journal_mode=WAL
Timeout 写锁等待超时，避免死锁 sqlite3.connect(timeout=10)

重复规则格式

类型格式示例
daily"HH:MM""09:00"
weekly"W HH:MM""5 17:00" 周五下午5点
monthly"D HH:MM""1 09:00" 每月1号9点
yearly"MM-DD HH:MM""12-25 10:00"

脚本功能

memo_cli.py

核心 CLI 工具，所有数据操作入口

  • add - 添加笔记
  • search - 搜索笔记
  • update - 更新笔记
  • delete - 删除笔记
  • remind - 添加提醒
  • due - 查询待触发提醒
  • dismiss - 废弃提醒
  • reminders - 查看提醒列表

reminder_scheduler.py

定时提醒调度器（cron 调用）

  • 每分钟由 cron 触发
  • 调用 memo_cli.py due
  • 通过 openclaw 推送消息

init.sql

数据库初始化脚本

  • 创建 notes 表
  • 创建 reminders 表
  • 创建 FTS5 全文索引
  • 创建触发器同步索引
  • 启用 WAL 模式

SKILL.md

AI 意图识别与操作指导

  • 定义触发词映射
  • 解析用户意图
  • 构造 CLI 命令
  • 处理返回结果
