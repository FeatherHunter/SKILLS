# 备忘录对话示例

## 1. 添加笔记
**用户**：“帮我记一下，张三的生日是10月3号，放社交夹里。”
**系统**：调用 `script/memo_cli.py add "张三生日10月3号" -c social`
**助手**：“已记下，放社交夹了。”

## 2. 带媒体的笔记
**用户**：“拍了一张购物小票，帮我存到备忘录，分类记账。”
**系统**：先处理图片文件加密保存到 `MEMO_MEDIA_DIR/slip_20260522.jpg`，然后调用 `script/memo_cli.py add "购物小票 2026-05-22" -c finance --media slip_20260522.jpg`
**助手**：“小票已存档，分类为记账。”

## 3. 搜索笔记
**用户**：“上次记的那个关于旅行愿望的笔记是什么来着？”
**系统**：调用 `script/memo_cli.py search "旅行" -c wish`
返回结果后，摘要展示。

## 4. 设置提醒
**用户**：“每周五早上9点提醒我交周报。”
**系统**：先添加笔记 `script/memo_cli.py add "交周报" -c work` 得到 id=10，
再 `script/memo_cli.py remind 10 --repeat-type weekly --rule "5 09:00"`
**助手**：“已设置每周五9点提醒交周报。”

## 5. 查看提醒
**用户**：“我有哪些提醒？”
**系统**：`script/memo_cli.py reminders` 列出所有 active 提醒。

## 6. 一次性提醒
**用户**：“提醒我明天下午3点打疫苗。”
**系统**：添加笔记 “打疫苗” → id=11，
`script/memo_cli.py remind 11 --at "2026-05-23 15:00"`
**助手**：“好的，明天下午3点提醒你。”

## 7. 删除笔记（级联删除提醒）
**用户**：“把那个‘交周报’的备忘删掉，不用提醒了。”
**系统**：先搜索找到 id=10，然后 `script/memo_cli.py delete 10`（自动删除关联提醒）。
**助手**：“已删除，提醒也取消了。”

## 8. 每日收尾仪式（成就记录）
**用户**：“今天搞定了数据库设计，记下来。”
**系统**：`script/memo_cli.py add "搞定了备忘录数据库设计" -c achievement`
**助手**：“成就已记录，今日收获+1。”
