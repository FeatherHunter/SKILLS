# 心理学原理科普

### 交互式学习原则

| 原则 | 说明 | 示例 |
|------|------|------|
| 采访式 | AI提问，用户思考回答 | "你觉得XX是怎么实现的？" |
| 诊断式 | 先测后学，发现薄弱点 | 预测试、阶段检测 |
| 验证式 | 用户先预测，再验证 | "先猜执行流程，再看源码验证" |
| 反思式 | 学习后用户总结 | "用自己的话总结要点" |

|------|------|---------|

| **近因效应** | 结尾总结强化 | 阶段4总结巩固 |

### 记忆原理总结

#### 短期记忆优化
- **组块化**：将复杂信息拆分成3-5个组块
- **主动回忆**：学完后立即回忆而非重读
- **双重编码**：文字+图示+示例

#### 长期记忆巩固
- **间隔重复**：1天、3天、7天、14天、30天
- **精细加工**：用类比、联想、视觉化
- **提取练习**：主动提取比重复学习更有效
- **变式练习**：多角度应用同一知识
