## 案例 1:抖音 vlog 封面(智剪工坊 DAY14)

**素材**:
- 主图:汉堡特写(主视觉)
- 左图:汉堡自拍
- 右图:健身房照片

**目标**:9:16 竖屏,3 张图对称层叠,带"14 天 -7 斤"钩子

```bash
封面合成 compose \
  --photos main.jpg left.jpg right.jpg \
  --layout symmetric-cascade \
  --aspect 9:16 \
  --text '{"main":"14 天","sub":"-7 斤","tags":"腰突 大基数"}' \
  -o cover_douyin.jpg
```

**预期输出**:9:16 画布,3 张图按 z=0/1/1 排列,文字"14 天"巨型居中,"-7 斤"橙红色上方,"腰突 大基数"底部标签行

## 诊断案例

### 半透明黑检测(DAY14 历史上踩过 4 次)

```bash
封面合成 diagnose cover.jpg --check transparency
```

如果 `pct > 1`,说明有半透明像素,通常是:
- rotate 用了 RGB 模式(应该用 RGBA + rotate_hard)
- alpha 羽化(应该二值化)
- shadow 扩散(应该去掉)

### 对称性检测

```bash
封面合成 diagnose cover.jpg --check symmetry
```

如果 `diff_pct > 8`,左右图位置没镜像。修正:
- 左图 `left_x` 和右图 `right_x` 应镜像于画布中心
- 1920 画布:左图左边缘 x1,右图右边缘 = 1920 - x1

### 整图过暗

```bash
封面合成 diagnose cover.jpg --check darkness
```

如果 `pct > 60`,整图过暗,可能:
- 画布色设错(应用 `#000000` 但不小心给了 #001122)
- 多个图层都是暗色 + 黑色背景

## 参数选择心法

### `layout` 选哪个?

| 场景 | 推荐 |
|---|---|
| 1 张主图 + 2 张副图,左右对称 | `symmetric-cascade`(默认) |
| 1 张主图 + 多张副图,营造"翻相册"感 | `polaroid` |
| 1 张主图 + 副图右下角堆叠,模拟"桌面贴纸" | `cascade` |
| 4+ 张图无主图,平铺展示 | `grid` |

### `aspect` 选哪个?

查 presets:
```bash
封面合成 presets --list
```

或直接传:
- 抖音 / 快手 → 9:16
- 视频号 → 3:4
- 小红书 → 4:5 / 1:1
- B 站 / YouTube → 16:9
- 微博 → 16:9

### `text` 怎么写?

**简单场景**(3 行文字):
```json
{"main": "主标题", "sub": "副标题", "tags": "标签1 · 标签2"}
```

**复杂场景**(自定义位置 / 字号):
```json
{
  "lines": [
    {"text": "大数字", "position": "middle-center", "size": 280},
    {"text": "副标题", "position": "top-center", "size": 100, "font_color": [255, 100, 50]},
    {"text": "标签行", "position": "bottom-center", "size": 50}
  ]
}
```

**位置**支持 9 宫格:`top-left` / `top-center` / `top-right` / `middle-left` / `middle-center` / `middle-right` / `bottom-left` / `bottom-center` / `bottom-right`

### 字号怎么选?

经验值(1920x1080 16:9 画布):
- 主标(钩子):200-300
- 副标:80-150
- 标签:40-70
- 装饰:20-40

如果你写"200"太撑,fit_text_to_area 会自动降字号(目前需手动调,后续做自动)
