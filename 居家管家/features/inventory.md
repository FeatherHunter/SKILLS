# 物品盘点

## 流程概述

1. 查询指定位置
2. 逐个确认物品状态
3. 用户反馈变更
4. 盘点完成

---

## Step 1: 查询位置

```bash
python home_manager.py inventory --location "卧室/衣柜"
```

---

## Step 2: 逐个确认

AI 逐个列出物品，请用户确认状态是否正确。

**注意**：同一物品可能同时出现在多个位置，盘点时要确认每个位置的状态。

---

## Step 3: 用户反馈变更

用户说"XX变了" → 进入「物品更新」处理。

支持用 `--location` 更新指定位置的数量：
```bash
# 某个位置的东西用了/少了
python home_manager.py update --id 1 --minus 1 --location "卧室/衣柜/上层"
```

---

## Step 4: 盘点完成

- 用户说"都对" → 结束盘点
- 中途退出：已确认的修改已写入，未确认的保持原样
