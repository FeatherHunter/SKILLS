# 看标签 / 合标签

## 流程概述

1. 列出所有标签
2. 识别相似标签
3. 确认并合并

---

## Step 1: 列出所有标签

```bash
python home_manager.py tag-list
```

---

## Step 2: 识别相似标签

AI 识别字面高度重叠的标签对，告知用户。

例如：
- "白" 和 "白色"
- "T桖" 和 "T恤"
- "牛奶" 和 "纯牛奶"

---

## Step 3: 确认并合并

用户确认 → 执行：

```bash
python home_manager.py tag-merge --from "白" --to "白色"
```

**注意**：合并后，原标签（"白"）的物品会自动更新为新标签（"白色"）。
