## 分类推断逻辑

AI 在解析用户输入时，根据关键词自动推断分类：
1. 精确匹配关键词 → 直接使用对应分类
2. 无法匹配 → 使用"其他"，回复中告知用户
3. 用户可在确认时纠正分类

## 各类 tag 必填/推荐维度（AI 录入参考）

AI 在 `录物品` / `拍物品` 流程中，确定 category 后按本表逐项询问 tag。
- **必填维度**：必须问到
- **推荐维度**：必须确认是否需要
- 标记 `*` 的维度 = **条件性**，AI 根据物品特性自行判定是否询问

### 通用 tag 维度（AI 心智模型）

录入时从 5 个方向思考 tag，不限具体维度：
- **谁用**：使用者（自己 / 家人 / 宠物）
- **什么样**：颜色、款式、材质、风格、新旧
- **哪来**：品牌、产地、购买渠道
- **多大/多重**：尺寸、容量、重量、规格、型号
- **何时/哪用**：季节、节日、使用场景、适用人群

### 条件性维度（`*`）AI 判定规则

标记 `*` 的维度，AI 必须先判定是否适用，再决定是否询问。

**双重判定标准**：

1. **能否回答**：用户看到这个维度能否给出具体值
   - 例：可口可乐 → "是否有机？"无法回答（加工饮料无此概念）
   - 例：鸡蛋 → "是否有机？"能回答（有机/普通）
2. **有无意义**：该维度的回答是否影响物品的辨识和管理
   - 例：可乐 → "是否有机"答"否"对管理无意义（所有可乐都加工）
   - 例：鸡蛋 → "是否有机"影响搜索（"找有机鸡蛋"）

**判定结果**：
- 两条都不通过 → 跳过，不计入 tag 总数
- 至少一条通过 → 询问并纳入

**常见判定示例**：

| 维度 | 不适用物品 | 适用物品 |
|------|-----------|---------|
| 食品-是否有机 | 可乐、薯片、罐头、酱油（深度加工） | 鸡蛋、蔬菜、有机牛奶 |
| 调味品-辣度 | 酱油、糖、盐（非辣） | 辣椒酱、火锅底料 |
| 化妆品-肤质 | 香水、彩妆（不区分肤质） | 面霜、精华、乳液 |
| 化妆品-香型 | 无香料产品 | 香水、香体喷雾 |
| 清洁用品-香型 | 消毒液、漂白剂（无香） | 洗衣液、洗洁精 |
| 文具-品牌 | 普通文具（塑料袋、便利贴） | 钢笔、专业工具 |
| 书籍-阅读状态 | 工具书、字典、参考手册 | 小说、教材 |
| 家电-功率 | 电视、音响（用屏幕尺寸/声道替代） | 空调、洗衣机、电饭煲 |
| 数码-是否无线 | 有线耳机、有线鼠标 | 蓝牙耳机、无线充电器 |
| 鞋帽-跟高 | 运动鞋、平底拖鞋 | 高跟鞋、靴子 |
| 衣物-性别 | 儿童服装、中性款式 | 男女区分明显的成人装 |
| 家具-风格 | 纯功能家具（货架、梯子） | 沙发、茶几、床 |
| 绿植-毒性 | 公认无毒的常见品种 | 滴水观音等需警示植物 |
| 数字资产-自动续费 | 一次性买断（软件授权） | 订阅制会员（视频网站） |

### AI 录入流程

1. 确定 category（参考上文"分类推断逻辑"）
2. 加载该分类的 tag 模板
3. **必填维度**：依次询问（颜色？品牌？规格？）
4. **推荐维度**：逐项判定
   - 普通维度 → 询问
   - 条件性维度（`*`）→ 双重判定（能否回答 + 有无意义），适用才问
5. **凑数策略**：若 tag 总数不足 10
   - 优先从通用 5 方向补充（如状态/新旧/使用场景）
   - 仍不足 → AI 主动询问"还有什么想标记的？"直到 10 个
6. 录入前展示完整 tag 清单让用户确认
