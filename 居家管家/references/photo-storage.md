# 照片存储配置指南

## 概述

居家管家技能支持灵活的照片存储配置，通过环境变量可以轻松管理照片文件的存储位置。

## 快速配置

### 1. 使用默认配置（推荐新手）

默认情况下，照片存储在技能目录下的 `photos/` 文件夹：

```
居家管家/
├── photos/           ← 照片自动存储在这里
│   ├── 1_物品名称.jpg
│   ├── 2_另一物品.jpg
│   └── ...
└── scripts/
```

**优点**：无需配置，开箱即用

### 2. 自定义存储路径

如果需要将照片存储到其他位置（如外部硬盘、云同步目录），可以设置环境变量：

#### Windows 设置方法

**方法一：临时设置（当前终端有效）**
```cmd
set HOME_PHOTOS_DIR=D:\MyPhotos\居家管家
```

**方法二：永久设置**
1. 右键"此电脑" → 属性 → 高级系统设置
2. 点击"环境变量"
3. 在"用户变量"中点击"新建"
4. 变量名：`HOME_PHOTOS_DIR`
5. 变量值：你的照片目录路径（如 `D:\MyPhotos\居家管家`）

#### Linux/Mac 设置方法

**方法一：临时设置**
```bash
export HOME_PHOTOS_DIR=/home/user/photos/home-manager
```

**方法二：永久设置**
将以下行添加到 `~/.bashrc` 或 `~/.zshrc`：
```bash
export HOME_PHOTOS_DIR=/home/user/photos/home-manager
```

然后执行：
```bash
source ~/.bashrc  # 或 source ~/.zshrc
```

## 照片命名规范

### 推荐格式

```
{物品ID}_{物品名称}.jpg
```

**示例**：
- `14_优瞳隐形眼镜盒.jpg`
- `57_USB数据线_usmile充电线.jpg`
- `100_冬季羽绒服.jpg`

### 命名规则

1. **物品ID**：数据库中的物品 ID（自动递增）
2. **分隔符**：使用下划线 `_` 连接
3. **物品名称**：简洁明了，去除特殊字符
4. **文件格式**：推荐 `.jpg`，也支持 `.png`

## 数据库存储

### 存储方式

- **数据库字段**：`items.photo`
- **存储内容**：相对路径（如 `14_优瞳隐形眼镜盒.jpg`）
- **完整路径**：`HOME_PHOTOS_DIR` + 相对路径

### 路径解析示例

```
环境变量：HOME_PHOTOS_DIR = D:\MyPhotos\居家管家
数据库值：photo = 14_优瞳隐形眼镜盒.jpg
完整路径：D:\MyPhotos\居家管家\14_优瞳隐形眼镜盒.jpg
```

## 常见使用场景

### 场景一：本地存储

```bash
# 使用默认配置
# 照片自动保存在 居家管家/photos/ 目录
```

### 场景二：外部硬盘

```bash
# 设置环境变量指向外部硬盘
set HOME_PHOTOS_DIR=E:\Backup\Photos\居家管家
```

### 场景三：云同步目录

```bash
# 使用 OneDrive/Dropbox/iCloud 同步目录
set HOME_PHOTOS_DIR=C:\Users\用户名\OneDrive\居家管家照片
```

### 场景四：多设备共享

```bash
# 使用 NAS 或共享文件夹
set HOME_PHOTOS_DIR=\\192.168.1.100\shared\居家管家照片
```

## 故障排除

### 问题 1：照片目录不存在

**症状**：测试显示"目录不存在"

**解决**：
1. 检查环境变量路径是否正确
2. 手动创建目录：`mkdir D:\MyPhotos\居家管家`
3. 或使用默认配置（会自动创建 `photos/` 目录）

### 问题 2：照片显示"完整路径"但文件不存在

**症状**：物品详情显示路径，但文件不在该位置

**解决**：
1. 检查文件是否被移动或删除
2. 确认环境变量是否与照片实际位置匹配
3. 使用文件管理器验证路径

### 问题 3：环境变量不生效

**症状**：设置了环境变量但程序仍使用默认路径

**解决**：
1. 重启终端/命令行窗口
2. 确认环境变量名称拼写正确（区分大小写）
3. 检查是否在正确的用户/系统级别设置

## 最佳实践

1. **定期备份**：照片目录应纳入备份计划
2. **统一格式**：所有照片使用相同格式（推荐 `.jpg`）
3. **及时整理**：添加物品后立即处理照片
4. **路径简洁**：避免路径过长或包含特殊字符
5. **文档记录**：记录环境变量配置，方便迁移
