A 扫描入库 从 OpenClaw session 文件提取消息和附件写入数据库
唤醒词场景命令状态
扫描消息 日常使用。基于 checkpoint 增量扫描，只处理新消息 python3 scripts/record.py 改名
全量扫描 数据修复。清空 checkpoint 重头扫描所有 session 文件 python3 scripts/record.py --full 新增
扫描附件 同扫描消息，附件随消息自动提取入库 python3 scripts/record.py 新增
B 查询消息 按时间、渠道、发送者等条件查询用户消息
唤醒词场景命令状态
查询消息 通用入口。需要额外指定日期或范围 python3 scripts/query.py --date YYYYMMDD 新增
查询日期 查某一天的全部消息 python3 scripts/query.py --date YYYYMMDD 新增
查询范围 查一段时间内的消息，如"本周一到周三" python3 scripts/query.py --start ... --end ... 新增
查询最近 快速查看最近 N 条消息，无需指定日期 python3 scripts/query.py --recent N 新增
查询今日 查看今天的消息 python3 scripts/query.py --date $(date +%Y%m%d) 改名
查询渠道 只看某个渠道的消息，如 QQ、微信、PC python3 scripts/query.py --channel qq/wechat/pc 新增
查询发送者 只看某个发送者的消息 python3 scripts/query.py --sender <id> 新增
C 查询附件 查询用户发送的图片、语音、文件等附件
唤醒词场景命令状态
查询附件 查看某日的所有附件列表 python3 scripts/query.py --date YYYYMMDD --attachments 新增
查询图片 只看图片附件（jpg/png/gif/webp） python3 scripts/query.py --date YYYYMMDD --type image 新增
查询语音 只看语音附件（wav/mp3） python3 scripts/query.py --date YYYYMMDD --type audio 新增
D 数据库管理 数据库初始化、状态查看、统计分析、索引维护
唤醒词场景命令状态
初始化数据库 首次使用或重建数据库，建表建索引 python3 scripts/status.py --init 新增
查看状态 查看 DB 路径、文件大小、各表记录数、checkpoint 概况 python3 scripts/status.py 新增
统计消息 查看消息总数、今日数、按渠道分布、近 7 天趋势 python3 scripts/status.py --stats 新增
重建索引 查询变慢时重建数据库索引优化性能 python3 scripts/status.py --reindex 新增
E 定时任务 管理录音机的 cron 定时扫描任务
唤醒词场景命令状态
查看定时 确认录音机 cron 任务是否在运行 openclaw cron list | grep 录音机 新增
创建定时 创建每 10 分钟自动扫描的定时任务 openclaw cron add --name "录音机-每10分记录语录" --every 10m ... 新增
删除定时 停止并删除定时任务 openclaw cron delete <task-id> 新增
手动触发 不等定时，立即执行一次扫描 openclaw cron run <task-id> 新增
