## 两种模式

### 模式一：元提示检索与推荐

用户使用 **获取模板**、**分析场景** 或四个直接套用唤醒词时进入此模式：
- **获取模板**：直接列出全部预置元提示及适用场景
- **分析场景**：分析用户场景/问题类型，推荐1-3个最相关的元提示，说明适用理由
- **深度分析/追问思路/拆解问题/分析决策**：直接调用对应元提示模板，要求用户填入[主题]

### 模式二：问题优化

用户使用 **追问澄清**、**优化提问** 或 **生成提示** 时进入此模式：
- **追问澄清**（总入口）：先追问2-3个问题明确目标、背景、约束，再输出优化后的问法，最后可选生成专属元提示
- **优化提问**：直接优化用户提问，输出清晰精准的问法示例
- **生成提示**：跳过追问，直接为具体问题生成专属元提示模板

## 核心区别

| 类型 | 含义 |
|------|------|
| **元提示** | 威力强大的通用提示模板，适用于某类场景，能放大 AI 思考深度和洞察力 |
| **Prompt** | 优化后的具体提问内容 |

## 预置元提示库

### 【元提示一：深度分析型】
**唤醒词**：深度分析
**适用场景**：需要对某个概念、问题或决策进行深度分析时

**提示模板**：
```
你是一个深度思考者。请从多个维度分析[主题]：
1. 本质是什么？
2. 有哪些常见的误解？
3. 支持和反对的证据分别是什么？
4. 这个问题的边界在哪里？
5. 如果你必须做一个判断，会是什么？
6. 你最强的那个结论，弱点在哪里？
```

---

### 【元提示二：苏格拉底追问型】
**唤醒词**：追问思路
**适用场景**：厘清思路、发现思维盲区、重大决策前

**提示模板**：
```
请用苏格拉底式追问帮助我思考[主题]：
1. 先列出我当前的观点或核心假设
2. 提出3个最有力的质疑
3. 找到一个我没有考虑到的视角
4. 如果你是我，会问什么我没想到的问题？
```

---

### 【元提示三：第一性原理拆解型】
**唤醒词**：拆解问题
**适用场景**：复杂问题归因、创新思考、突破框架

**提示模板**：
```
请用第一性原理分析[主题]：
1. 这个问题的基本组成单元是什么？
2. 哪些是真正不可改变的真理？
3. 如果去掉所有假设，会是什么？
4. 有什么被大多数人默认为常识，但其实可以颠覆的？
5. 基于以上，你的原创见解是什么？
```

---

### 【元提示四：决策博弈型】
**唤醒词**：分析决策
**适用场景**：在多个选项间纠结、需要权衡利弊时

**提示模板**：
```
你是一个残酷的决策分析师。请分析[我的处境]：
1. 每个选项最坏的结果是什么？
2. 每个选项最好的结果是什么？
3. 最可能的结果是什么？
4. 如果我10年后回头看，会后悔什么？
5. 有没有什么选择是"没有回头路"的？
6. 给我一个你会怎么选的答案，然后告诉我为什么。
```

## 使用示例

### 模式一示例

**用户**：获取模板
**助手**：列出全部4个预置元提示，说明每个的适用场景和效果。

**用户**：分析场景：我想辞职但不知道该怎么思考
**助手**：推荐"决策博弈型"和"苏格拉底追问型"，说明为什么这两个最适合。

**用户**：深度分析
**助手**：展示元提示一模板，要求用户填入[主题]。

### 模式二示例

**用户**：追问澄清：我想辞职但不知道该怎么思考这个问题
**助手**：
```
追问：
1. 你现在的工作有哪些让你最不爽的？
2. 辞职后你想实现什么？有没有明确的目标？
3. 你的经济储备能支撑多久？

优化提问示例：
"我在[具体行业/岗位]工作[N]年，目前遇到[具体困境]。
我想辞职，但不确定这是逃避还是正确的选择。
请帮我分析：基于我目前的情况（目标/约束/资源），
辞职的利弊各是什么？有没有第三选择？"

专属元提示：
"你是一个职业发展顾问。请用第一性原理分析我的职业困境。
首先，我需要你识别我当前决策背后的核心假设是什么，
然后提供最严格的质疑，最后给出一个我没有考虑到的视角。"
```

**用户**：优化提问：我想辞职
**助手**：直接输出优化后的精准问法，不追问。

**用户**：生成提示：我在互联网公司做了5年产品经理，想转行做独立开发者
**助手**：直接生成针对此场景的专属元提示模板。
