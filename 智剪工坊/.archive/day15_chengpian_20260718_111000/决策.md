
## 反模式 (踩过的坑)

| 坑 | 解决 |
|---|---|
| PowerShell 客户端 120s 超时杀掉 ffmpeg | 用 600s timeout + background 模式 |
| ffmpeg alpha 表达式太长报错 (drawtext fade) | `--no-shadow` 禁用 fade in/out 表达式 |
| cover.py --title-main 只能塞 1 行 | 切到 PIL 直接画(canvas + draw + font) |
| cover 默认 9:16,user 要 16:9 | PIL 重新画 1920×1080 |
| mavis-trash 拒绝 reparse point | 用真实 D:\ 路径,不写 C:\Users\... |
| 文字一行超出安全区 | 字号缩到 220 / 100 / 56 |
