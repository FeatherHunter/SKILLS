# assets/

这里放**资源文件**(LUT、字体、测试视频等)。

## 建议目录

```
assets/
├── luts/              电影级 LUT(.cube 文件)
│   ├── cinematic.cube
│   ├── teal_orange.cube
│   └── vintage.cube
├── fonts/             开源字体(Noto Sans CJK 等)
└── test_videos/       测试用样片(空文件夹)
```

## LUT 来源

免费的 LUT 包:
- https://fixthephoto.com/free-luts.html
- https://www.rocketstock.com/free-luts-for-premiere-pro/
- https://luts.io/

下载后放入 `assets/luts/`。

## 字体

优先开源字体(避免版权问题):
- **Noto Sans CJK** —— Google 开源,覆盖中/日/韩
- **思源黑体** —— Adobe + Google
- **Source Han Sans**

下载后放入 `assets/fonts/`。
