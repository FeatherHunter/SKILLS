转录 → agent 改写文案 → matrix TTS 327 声音合成 → ffmpeg 替换音轨。剪映做不到 word-level 改写。

327 TTS 声音 自由改写 音轨替换
查看命令 
# 1. transcribe
python scripts/rewrite_audio.py transcribe \
  --input v.mp4 --srt v.srt

# 2. (Mavis 改写文案,选 voice_id)
# 3. synthesize
python scripts/rewrite_audio.py synthesize \
  --text "改写后的文案" \
  --voice male-qn-jingying \
  --out v_new.mp3

# 4. replace
python scripts/rewrite_audio.py replace \
  --video v.mp4 --audio v_new.mp3 \
  --out v_final.mp4

实测:20s → 11.8s(省 41% 时长)

FAQ

常见问题

30 个高频 Q&A,看这里大概率能解决。

智剪工坊 vs 剪映,我该用哪个?

用智剪工坊:批量处理 100+ 视频 / AI 能力定制(去水词 / 改词) / CLI 自动化(跑定时任务) / 帧级精度。

用剪映:单条精修(关键帧调色) / 所见即所得 / 模板丰富 / 声音克隆 / 移动端。

最佳组合:智剪工坊做粗活(批量 + 去水词 + 改词) → 剪映做精修(关键帧 + 声音克隆)。

需要 API key 或 GPU 吗?

不需要 API key。所有 AI 能力走 mmx matrix MCP,免 key,免 GPU。

GPU 推荐但非必需:有 GPU 时 Whisper 转录快 5-10 倍。CPU 也能跑(small 模型)。

视频时长变 8 小时 / 没声音 / 进度条卡死 怎么办?

8 小时:不同原始视频 fps 不一致。已用 fps=30 统一 filter 修复。

没声音:ffmpeg 没传 -c:a aac(本工具默认已加)。

NVENC 崩溃(Access Violation):用 libx264(本工具默认)。

AI 中文乱码:生图对中文差。已用"先生视觉 + 后叠中文"两步法。

怎么在 Mavis agent 中调用?

直接说自然语言,小帅(本 skill)会自动路由到对应子技能:

你: "把 vlog 切到 1 分钟,然后加 BGM,然后去水词"
小帅:
  → cut.py trim (切 60s)
  → bgm_loop.py (加 BGM)
  → remove_fillers.py (去水词)
  → 输出成片
