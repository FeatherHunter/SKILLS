# 查看食谱 + 做菜模式

> 触发词："看看宫保虾球怎么做"、"查看食谱"、"菜谱详情"、"XX的完整步骤"、"开始做这道菜"、"做菜模式"

---

## 功能说明

### 查看食谱
用户想了解一道菜的做法时，显示完整信息。

### 做菜模式
从查看食谱延伸，在做菜前提供：
- 需要厨具清单
- 需要食材清单
- 完整步骤（检查清单）

用户说"开始做这道菜"或"做菜模式"时触发。

---

## 查看食谱

### 输出格式

```
【宫保虾球】

基本信息
- 难度：中等
- 总时间：25分钟
- 份量：2人份
- 状态：已做（做过3次，评分4.5）
- 来源：中餐厅节目

分类标签
- 菜系：川菜
- 地区：中国-四川
- 季节：春/秋
- 烹饪方式：炒
- 口味：辣/麻
- 用餐类型：中餐/晚

需要的厨具
- 炒锅
- 漏勺

食材清单（共8种）
1. 虾 300g
2. 花生 50g（可选）
3. 干辣椒 10g
4. 花椒 5g
5. 葱 1段
6. 蒜 3瓣
7. 盐 适量
8. 料酒 15ml

完整步骤
第1步（10分钟）：虾去壳开背，用料酒和盐腌制10分钟
第2步（3分钟）：大火热油，虾下锅炸至变色捞出
第3步（1分钟）：锅留底油，大火爆香花椒和干辣椒
...

技法关联
- 第2步：油炸（高油温快速定型）
- 第3步：爆炒（大火激发香气）

小贴士
- 第2步火候：大火，油温要够
- 虾的处理：开背去虾线更入味

背景故事
宫保虾球源自川菜宫保鸡丁的变体...

营养信息（每份200g）
- 热量：320kcal
- 蛋白质：28g
- 脂肪：18g
```

## 做菜模式（检查清单）

### 触发方式
- "开始做这道菜"
- "做菜模式"
- "开始做宫保虾球"

### 输出格式

```
【做菜模式 - 宫保虾球】

需要厨具
□ 炒锅
□ 漏勺
□ 砧板

需要食材（检查一下你有没有）
□ 虾 300g
□ 花生 50g（可选）
□ 干辣椒 10g
□ 花椒 5g
□ 葱 1段
□ 蒜 3瓣
□ 盐 适量
□ 料酒 15ml

完整步骤
第1步（10分钟）：虾去壳开背，用料酒和盐腌制10分钟
第2步（3分钟）：大火热油，虾下锅炸至变色捞出
第3步（1分钟）：锅留底油，大火爆香花椒和干辣椒
...

小贴士
- 虾开背时去虾线，更入味
- 油温不够会导致虾吸油

【下一步 - 未来设计】
逐步指导（说"下一步"逐步引导做菜）→ 未来版本设计
```

## 查询逻辑

1. 先通过菜名查 `recipes` 表获取 `recipe_id`
2. 用 `recipe_id` 查所有关联表
3. 按格式组装输出
