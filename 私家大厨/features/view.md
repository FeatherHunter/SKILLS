# 查看食谱 + 做菜模式

> 触发词："看看XX怎么做"、"查看食谱"、"菜谱详情"、"XX的完整步骤"、"开始做这道菜"、"做菜模式"

---

## 功能说明

### 查看食谱
用户想了解一道菜的做法时，显示完整信息。

### 做菜模式
从查看食谱延伸，在做菜前提供：
- 需要厨具清单
- 需要食材清单
- 完整步骤（检查清单）

用户说"开始做这道菜"或"做菜模式"时触发。

---

## AI调用规范

调用 `show` 命令时，输出完整的食谱信息，包括：
- 主表所有字段
- 所有关联表的完整数据
- 步骤×食材关联
- 步骤技法

如果某字段为空，显示为"未知"或"-"。

## 查看食谱

### 输出格式

```
【宫保虾球】

基本信息
- 难度：中等
- 总时间：25分钟
- 份量：2人份
- 状态：已做（做过3次，评分4.5）
- 来源：中餐厅节目
- 照片：[URL或路径]

分类标签
- 菜系：川菜
- 地区：中国-四川
- 国家：中国
- 季节：春/秋
- 烹饪方式：炒
- 口味：辣/麻
- 饮食标签：荤菜
- 用餐类型：中餐/晚

需要的厨具
- 炒锅（锅）
- 漏勺

食材清单（共8种）
1. 虾 300g | 可选：否
2. 花生 50g | 可选：是（可用腰果代替）
3. 干辣椒 10g
4. 花椒 5g
5. 葱 1段
6. 蒜 3瓣
7. 盐 适量
8. 料酒 15ml

完整步骤
第1步（10分钟）[小火] 温度：常温
  动作：虾去壳开背，用料酒和盐腌制10分钟
  预期：虾肉变红，去腥
  投入：虾 300g（开局加入）
  技法：腌制（用料酒去腥，关键：时间要够/料酒要适量）

第2步（3分钟）[大火] 温度：180度
  动作：大火热油，虾下锅炸至变色捞出
  预期：虾肉变红，表面微焦
  投入：虾 300g（第2步加入）
  技法：油炸（高油温快速定型，关键：油温要高/炸制时间短）

第3步（1分钟）[大火] 温度：200度
  动作：锅留底油，大火爆香花椒和干辣椒
  预期：香气四溢，花椒辣椒变色
  投入：花椒 5g | 干辣椒 10g（开局加入）
  技法：爆炒（大火激发香气，关键：火要大/时间要短）

第4步（2分钟）[大火] 温度：200度
  动作：放入虾球和花生，快速翻炒均匀，出锅
  预期：虾球裹满酱汁，花生酥脆

小贴士
- [第1步-刀工] 开背时去虾线更入味（优先级：1）
- [第2步-火候] 油温不够会导致虾吸油（优先级：1）
- [采购] 选择新鲜活虾，口感更Q弹（优先级：2）
- [荸荠] 是经典配料，增加脆嫩口感，也可用莲藕替代（优先级：2）

背景故事
宫保虾球源自川菜宫保鸡丁的变体，由山东人丁宝桢发明...

营养信息（每份200g）
- 热量：320kcal
- 蛋白质：28g
- 脂肪：18g
- 碳水：20g
- 膳食纤维：2g
- 钠：800mg

烹饪历史
- 2026-05-15 第1次 评分4.5「味道不错，虾很Q弹」
```

## 做菜模式（检查清单）

### 触发方式
- "开始做这道菜"
- "做菜模式"
- "开始做宫保虾球"

### 输出格式

```
【做菜模式 - 宫保虾球】

需要厨具
□ 炒锅（锅）
□ 漏勺
□ 砧板

需要食材（检查一下你有没有）
□ 虾 300g
□ 花生 50g（可选，可用腰果代替）
□ 干辣椒 10g
□ 花椒 5g
□ 葱 1段
□ 蒜 3瓣
□ 盐 适量
□ 料酒 15ml

完整步骤
第1步（10分钟）：虾去壳开背，用料酒和盐腌制10分钟
第2步（3分钟）：大火热油，虾下锅炸至变色捞出
第3步（1分钟）：锅留底油，大火爆香花椒和干辣椒
第4步（2分钟）：放入虾球和花生，快速翻炒均匀，出锅

小贴士
- 第1步刀工：开背时去虾线，更入味
- 第2步火候：油温不够会导致虾吸油

【下一步 - 未来设计】
逐步指导（说"下一步"逐步引导做菜）→ 未来版本设计
```

## 查询逻辑

1. 先通过菜名查 `recipes` 表获取 `recipe_id`
2. 用 `recipe_id` 查所有关联表
3. 按格式组装输出

---

## 命令参考

```bash
# 查看食谱详情（完整输出）
python scripts/recipe_manager.py show <菜名或ID>

# 查看某食谱的步骤
python scripts/step_manager.py list <recipe_id>

# 查看某食谱的技法
python scripts/technique_manager.py list-by-recipe <recipe_id>

# 查看某食谱的小贴士
python scripts/tip_manager.py list <recipe_id>

# 查看某食谱的背景知识
python scripts/background_manager.py get <recipe_id>

# 查看某食谱的营养信息
python scripts/nutrition_manager.py get <recipe_id>

# 查看某食谱的烹饪历史
python scripts/history_manager.py list <recipe_id>

# 查看某食谱的炊具
python scripts/cookware_manager.py list <recipe_id>
```
