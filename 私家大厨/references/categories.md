## 难度等级

| 值 | 说明 |
|----|------|
| 快手菜 | 10分钟以内搞定 |
| 简单 | 15-30分钟，简单步骤 |
| 中等 | 30-60分钟，有一定技巧要求 |
| 困难 | 60分钟以上，复杂技巧或多个步骤 |
| 大师 | 需要专业技能或丰富经验 |
