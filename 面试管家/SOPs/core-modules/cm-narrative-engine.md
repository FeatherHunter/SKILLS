# 叙事引擎对接模块

> 模块：cm-narrative-engine  
> 用途：调用叙事引擎生成叙事包  
> 依赖：面试备战叙事引擎.md, learning-system/progress.json

---

## 一、模块概述

本模块对接"面试备战叙事引擎"，按轮次生成叙事包供用户提前练习。

---

## 二、叙事包类型

| 轮次 | 叙事包 | 内容 |
|------|--------|------|
| 一面 | 技术广度概览 | 基础概念 + 1-2个STAR案例 |
| 二面 | 主线项目故事 | STAR案例 + 追问弹药库 |
| 三面 | 架构视野陈述 | ADR + 技术趋势 |
| 四面 | 成长故事线 | 时间线 + 团队影响力 |
| 五面 | 战略视野 | 前瞻性 + 价值观 |

---

## 三、调用流程

```
1. 接收主流程传入的参数：
   - round: 当前轮次
   - target: {company?, position?}

2. 读取learning-system/progress.json：
   - interview_assets路径
   - 当前Level

3. 按round选择叙事包类型

4. 调用叙事引擎生成叙事包（内存结构）

5. 返回叙事包给轮次流程
```

---

## 四、返回格式

```json
{
  "round": "一面",
  "main_story": {
    "title": "Kotlin协程在项目中的应用",
    "star_case": "..."
  },
  "anchor_points": [
    "我研究过协程的状态机实现...",
    "在项目中遇到的生产问题..."
  ],
  "weakness_alerts": [
    "状态机源码细节不熟"
  ]
}
```

---

## 五、使用场景

- 轮次流程Step 0：加载叙事包给AI面试官参考
- 面试后更新叙事包：哪些追问答得好、哪些答不出

---

> 本模块由轮次SOP调用
